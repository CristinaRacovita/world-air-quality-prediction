The data set called World Air Quality Index by City and Coordinates aims to provide valuable insights about the air quality of different regions. The goal of my project is to classify the air quality of a given city (Good, Moderate, etc.). This is predicted using the country and city name, carbon monoxide, ozone, PM2.5 and nitrogen value. The data set is described in the first section.

Motivation Behind Model Choice

I chose a Generalized Linear Model (GLM) because it offers a good mix of simplicity and performance, making it ideal for this project. Since the target variable is the Air Quality Index Category, which is a classification problem, GLM is a natural fit. It helps understand how various pollutants, such as CO, Ozone, NO₂, and PM2.5, influence air quality in different regions.

Additionally, with a dataset of around 16k rows, GLM handles the data efficiently without needing complex adjustments or tuning like other models. This ensures the analysis remains straightforward while delivering reliable and interpretable predictions.

The data set has 14 columns and 16695 rows. The features are Country, City, AQI Value, AQI Category, CO AQI Value, CO AQI Category, Ozone AQI Value, Ozone AQI Category, NO2 AQI Value, NO2 AQI Category, PM2.5 AQI Value, PM2.5 AQI Category, lat, lng, and they are described as follows:

  • Country: Country Name
  • City: City Name
  • AQI Value/AQI Category: Air Quality Index
  • CO AQI Value/CO AQI Category: Carbon Monoxide (CO) is a colorless, odorless gas that is produced by the incomplete burning of fossil fuels. High levels of CO can be toxic to humans and can cause headaches, dizziness, and nausea.
  • Ozone AQI Value/Ozone AQI Category: Ozone is a gas that can form in the atmosphere through a chemical reaction between sunlight and other pollutants. High levels of ozone can be harmful to human health, particularly for those with respiratory issues.
  • NO2 AQI Value/NO2 AQI Category: Nitrogen Dioxide (NO₂) is a gas that forms in the atmosphere from the combustion of fossil fuels, mainly from vehicles and industrial activities. Elevated levels of NO₂ can be harmful to human health, particularly for individuals with respiratory conditions, as it can irritate the airways and worsen lung diseases like asthma.
  • PM2.5 AQI Value/PM2.5 AQI Category: PM2.5 refers to tiny particles or droplets in the air that are 2.5 micrometers or less in width. They can be harmful to human health when inhaled, especially in high concentrations.
  • lat: Region Latitude
  • lng: Region Longitude

For the above mentioned goal, the target column will be AQI Category; therefore, during the data cleaning, the feature AQI Value should be removed. This approach encourages the model to focus on other relevant features that contribute to air quality classifications, enhancing its ability to generalize and make accurate predictions based on patterns in the data rather than relying on direct mappings or redundant information.

Feature Selection

set.seed(194)

folds <- createFolds(dataset$Air_Quality_Index_Category, k = 10)

precisions <- c()
recalls <- c()
feature_counts <- table(character())

for (fold_index in 1:length(folds)) {
  test_fold <- as.data.frame(dataset[folds[[fold_index]], ])
  train_fold <- as.data.frame(dataset[-folds[[fold_index]], ])
  rf_model <-  rfsrc(Air_Quality_Index_Category ~ ., data = train_fold, importance = TRUE, ntree = 1)
  
  importance_scores <- rf_model$importance[, "all"]
  mean_importance <- mean(importance_scores)
  selected_features <- names(importance_scores[importance_scores >= mean_importance])
  
  fold_features <- table(selected_features)
  feature_counts <- table(c(names(feature_counts), names(fold_features)))
  
  selected_features_string = paste(selected_features, collapse = "+")
  formula = as.formula(paste("Air_Quality_Index_Category ~", selected_features_string))
  final_rf_model <- rfsrc(formula, data = train_fold, ntree = 1)
  
  predictions = predict(final_rf_model, test_fold)
  confusion_matrix <- confusionMatrix(factor(predictions$class), factor(test_fold$Air_Quality_Index_Category))
  precisions <- append(precisions, confusion_matrix$byClass[,"Pos Pred Value"])
  recalls <- append(recalls, confusion_matrix$byClass[,"Sensitivity"])
}

mean_precision <- mean(precisions, na.rm = TRUE)
mean_recall <- mean(recalls, na.rm = TRUE)
f1_score <- 2 * (mean_precision * mean_recall) / (mean_precision + mean_recall)
sorted_features <- sort(feature_counts, decreasing = TRUE)

print(paste("F1 Score:", f1_score))
